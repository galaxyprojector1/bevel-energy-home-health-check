# Bevel Energy - Bilan Sante Maison

Formulaire interactif pour analyser l'efficacite energetique d'une maison.

## Stack
- React 18
- Vite
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons

## Installation

```bash
npm install
npm run dev
```

## Deploiement Vercel

Ce projet est configure pour etre deploye automatiquement sur Vercel.
